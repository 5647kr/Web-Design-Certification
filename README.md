# Web-Design-Certification

## 웹디자인기능사 자격증 실기 개요
24년도 웹디자인기능사 자격증 취득을 위한 실기 연습 repository로 지금까지 출제한 (2024년도 F유형까지) 모든 유형을 직접
코딩하며 제작하였다. 01~ 10까지는 시중에 판매하는 `이기적 웹디자인기능사 실기`를 토대로 작성하였으며 11 ~ 34까지는 A유형부터 F유형까지 작업한 사이트이다.
